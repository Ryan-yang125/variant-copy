# Privacy Policy

Variant Copy is a Chrome/Edge extension for copying HTML source from Variant community cards.

## Data Collection

Variant Copy does not collect, store, sell, or share personal information.

Variant Copy does not use analytics, tracking pixels, advertising identifiers, remote logging, or a developer-operated backend.

## Data Processed Locally

When you open the toolbar popup, Variant Copy checks whether the active tab is `https://variant.com/community` so it can show a simple status message.

When you click a card copy button, Variant Copy reads that card's `data-design-id`, fetches the matching HTML file from `https://variant.com/design/<id>.html`, and writes that HTML text to your clipboard.

## Network Requests

Variant Copy makes network requests only to `https://variant.com/design/*.html`, and only after you click a copy button on a Variant community card.
